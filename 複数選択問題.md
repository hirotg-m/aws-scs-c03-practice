# AWS認定　セキュリティ・専門知識

## 概要

- AWS認定　セキュリティ専門知識の模擬試験の内容を復習しています
- 問題について、正しい選択はどれですか



## 問題
金融サービス企業が、複数のWindows EC2インスタンスからのトランザクションログを、各サーバーにインストールされたCloudWatch Logsエージェントを通じてAmazon CloudWatchでモニタリングしています。エージェントの設定ファイルは検証済みで、送信されるべきログファイルも正しく設定されています。しかし、コンプライアンス担当者が、いくつかのEC2インスタンスがまったくログを送信していないことを発見しました。この問題を調査し解決するために、どのような手順を取るべきですか？（2つ選択）

## 選択1
AWS X-Rayを使用して、CloudWatch Logsエージェントのパフォーマンスをトレースおよび診断します。

## 選択2
AWS Systems Manager Run Commandを使用して、すべてのAmazon EC2インスタンスでawslogsサービスが稼働していることを確認します。

## 選択3
C:\CloudWatch\rejects.logファイルを調べて、無効なタイムスタンプや破損したデータによって拒否されたトランザクションログエントリを特定します。

## 選択4
CloudWatch Logsエージェントが使用するIAM権限が、ロググループとログストリームの作成、およびCloudWatchにログイベントを配置する能力を許可しているかを確認します。

## 選択5
CloudWatchで拡張モニタリングを有効にして、より詳細なメトリクス収集を行います。
